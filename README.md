# S6SDK-iOS

1.将S6SDK.framework拉入项目

2.将Thirdparty文件夹拉入项目

3.项目设置framework search path 中添加Thirdparty文件夹

4.添加以下依赖

- CoreTelephony.framework
- libz.tbd 
- libsqlite3.tbd
- SystemConfiguration.framework
- Security.framework
- libc++.tbd
